# 고고학 발굴조사보고서 고찰 작성 Skills

한국 고고학 발굴조사보고서의 "고찰" 섹션을 효율적으로 작성하기 위한 AI 지원 Skill 시스템입니다.

## 개요

이 프로젝트는 Claude Code를 사용하여 고고학 발굴조사보고서의 고찰 섹션을 학술 논문 수준으로 작성할 수 있도록 지원하는 8개의 전문 Skill을 제공합니다. 각 Skill은 고찰의 특정 항목에 특화되어 있으며, 다양한 입력 형식(Excel, CSV, Markdown, JSON)을 지원합니다.

## Skill 목록

| 번호 | Skill명 | 항목명 | 설명 |
|------|---------|--------|------|
| 01 | `gochal-01-conclusion` | 결론 및 종합 분석 | 모든 고찰 내용을 종합하여 유적의 성격과 학술적 의의 규명 |
| 02 | `gochal-02-yugu-analysis` | 유구 분석 | 유구의 형태, 규모, 특징 분석 및 시공간적 관계 해석 |
| 03 | `gochal-03-yumul-analysis` | 유물 분석 | 유물의 종류, 형식, 특징 분류 및 문화적 의미 해석 |
| 04 | `gochal-04-pyeonnyeon` | 편년(編年) | 형식학적 분석과 과학적 연대측정을 통한 시기 설정 |
| 05 | `gochal-05-site-comparison` | 주변 유적과의 비교 | 주변 유적과의 비교를 통한 지역적·문화적 맥락 파악 |
| 06 | `gochal-06-science-analysis` | 자연과학 분석 종합 | AMS 연대, 고환경 분석 등 자연과학 분석 결과 종합 |
| 07 | `gochal-07-culture-restoration` | 문화 양상 및 생활상 복원 | 주거 생활, 생업 경제, 사회 조직, 정신 문화 복원 |
| 08 | `gochal-08-summary` | 요약 및 초록 | 조사 내용 요약 및 한글·영문 초록 작성 |

## 주요 기능

### 1. 다양한 입력 형식 지원
- Excel (.xlsx, .xls)
- CSV (.csv)
- Markdown (.md)
- JSON (.json)
- 사용자 직접 입력

### 2. 학술 논문 수준 출력
- 고고학 전문 용어 사용
- 표 및 통계 데이터 포함
- 참고문헌 형식 준수
- 학술적 문체

### 3. 청동기시대·삼국시대 특화
- 전남 지역 유적 중심
- 실제 보고서 예시 기반
- 지역별 특성 반영

## 사용 방법

### 기본 사용 예시

1. **유구 분석 고찰 작성**
   ```
   사용자: "나주 복암리 유적의 유구 분석 고찰 작성해줘"
   ```
   - `gochal-02-yugu-analysis` skill이 자동 트리거
   - 유구 데이터 파일 요청
   - Markdown 형식으로 고찰 생성

2. **편년 고찰 작성**
   ```
   사용자: "편년 고찰 작성해줘. AMS 연대측정 결과 포함해서."
   ```
   - `gochal-04-pyeonnyeon` skill이 자동 트리거
   - AMS 연대 데이터 입력
   - 상대편년 + 절대편년 통합 고찰 생성

3. **전체 고찰 종합**
   ```
   사용자: "결론 및 종합 분석 작성해줘"
   ```
   - `gochal-01-conclusion` skill이 자동 트리거
   - 앞선 고찰 내용 종합
   - 최종 결론 생성

## 설치 방법

이 Skills는 Claude Code의 `.claude/skills/` 디렉토리에 이미 설치되어 있습니다. 별도의 설치 과정은 필요하지 않습니다.

## 입력 데이터 예시

### Excel 파일 (유구목록)
| 번호 | 형태 | 크기 | 구성요소 | 출토유물 |
|------|------|------|----------|----------|
| 주거지1 | 방형 | 4.5×4.2m | 노지,저장공 | 무문토기,석촉 |
| 주거지2 | 원형 | 직경 3.8m | 노지 | 무문토기편 |

### JSON 데이터
```json
{
  "site_info": {
    "name": "나주 복암리 3지구",
    "period": "청동기시대",
    "region": "나주"
  },
  "data": {
    "유구목록": [
      {
        "번호": "주거지1",
        "형태": "방형",
        "크기": "4.5m × 4.2m",
        "구성요소": ["노지", "저장공"],
        "출토유물": ["무문토기", "석촉"]
      }
    ]
  }
}
```

## 출력 예시

```markdown
## 2. 유구 분석

### 2.1 주거지

조사 결과 총 15기의 주거지가 확인되었다. 주거지는 평면 형태에 따라 
방형(10기), 원형(3기), 부정형(2기)으로 분류된다.

**표 1. 주거지 규모 현황**
| 구분 | 개수 | 평균 크기 | 비고 |
|------|------|-----------|------|
| 방형 | 10기 | 4.2×4.0m | 청동기시대 전기 |
| 원형 | 3기 | 직경 3.5m | 청동기시대 중기 |

방형 주거지는 장축 방향이 남북 방향으로 배치되어 있으며...
```
